# IST495 Real-Time Stock Dashboard

## Overview
This project analyzes real-time stock sentiment and market activity using data from Stocktwits and Finviz.

## Features
- Real-time sentiment tracking
- Rumor detection per ticker
- Interactive dashboard (Streamlit)
- Time-based analysis and visualization

## Tech Stack
- Python
- MongoDB
- Streamlit
- Pandas
- Plotly

## How to Run
1. Run scraper
2. Run finviz fetch
3. Launch dashboard:
   python -m streamlit run streamlit_app/app.py
